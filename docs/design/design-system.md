# Design System - Adaptive Productive

## Philosophy

**"Productivity that learns your rhythm"** — Modern, student-friendly, calm but energetic. Avoid corporate/generic aesthetics.

## Color Palette

| Token | Hex | Use |
|-------|-----|-----|
| Primary | `#0D9488` | Actions, links, focus states |
| Primary Light | `#0F766E` | Hover/pressed |
| Accent | `#F97316` | AI features, highlights |
| Accent Light | `#FED7AA` | Backgrounds |
| Background | `#FAFAF9` | App background |
| Surface | `#FFFFFF` | Cards, modals |
| Text | `#1C1917` | Primary text |
| Text Secondary | `#57534E` | Secondary text |
| Muted | `#A8A29E` | Placeholders, disabled |
| Border | `#E7E5E4` | Dividers |
| Success | `#22C55E` | Completed, success states |

## Typography

- **Title**: 28px, bold (700), -0.5 letter-spacing
- **Subtitle**: 18px, semibold (600)
- **Body**: 16px, regular (400)
- **Caption**: 14px, regular, secondary color
- **Small**: 12px, medium (500)

## Spacing

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px

## Radius

- sm: 8px
- md: 12px
- lg: 16px
- xl: 24px
- full: 9999px (pill)

## Components

- **Cards**: White surface, 12px radius, subtle shadow
- **Buttons**: Primary = teal fill; Secondary = accent fill; Text = muted
- **Checkbox**: 24px circle, 2px border, primary color when done
